# integration-field-output（集成需求字段清单产出）

本目录用于存放 AI 输出物模板文件。

## 所需文件

请将以下 Excel 文件放置在此目录：

### AI输出物摸板.xlsx

**文件名**: `AI输出物摸板.xlsx`

**文件结构**（包含以下标准工作表）：

| 工作表名称 | 说明 |
|-----------|------|
| 集成配置 | 集成接口配置信息模板 |
| 开发计划 | 开发计划和排程模板 |
| 问题跟进管制-TB导入联动 | 问题跟踪和管制模板 |

## 文件用途

导出 API 匹配结果时，会自动从模板中复制这些标准工作表到输出文件中，确保输出格式符合规范要求。

**输出文件结构**：
1. **输入物-TB导入EXCEL摸板** - 需求匹配结果（程序自动生成）
2. **集成配置** - 从模板复制
3. **开发计划** - 从模板复制
4. **问题跟进管制-TB导入联动** - 从模板复制
5. **字段对照详情** - 匹配到的接口字段对照工作表

## 文件获取方式

1. 从团队共享位置获取标准模板文件
2. 或参考 skill 脚本中的示例模板创建

## 配置说明

后端配置文件 (`music-server/src/main/resources/application-dev.properties`) 中的相关配置：

```properties
# AI 输出物模板路径
api.match.output-template-dir=integration-field-output
api.match.output-template-name=AI输出物摸板.xlsx
```

## 注意事项

1. 如果模板文件不存在，导出功能仍可正常工作，但不会包含标准工作表
2. 模板文件更新后，无需重启服务即可生效
3. 建议保持模板格式与团队规范一致
